🐾 Pet System Demo

🎯 What is the Pet System?

The Pet System is a Tamagotchi-like experience integrated into your typing trainer! As you complete typing tests, your pet will grow, evolve, and respond to your performance.

🌟 8 Evolution Stages

Your pet starts as an egg and evolves through 8 unique stages, each with different appearances and abilities.

🎮 Interactive Features

Feed your pet, play with it, and watch it respond to your typing performance with animations.

📊 Performance Tracking

Your pet's stats are directly tied to your typing performance - accuracy, speed, and consistency matter!

🎨 Canvas-Based Animation

Built with HTML5 Canvas for smooth, sprite-based animations with walking, idle, happy, and sad states.

🎮 How It Works

The pet system integrates seamlessly with your typing tests:

// When you complete a typing test: const result = { wpm: 60, accuracy: 98, errors: 2, // ... other test data }; // Your pet gains experience and happiness petManager.updatePetFromTestResult(result); // Good performance = happy pet + experience gain // Poor performance = sad pet + slower growth

📈 Experience Gain

WPM Bonus: Up to 20 exp based on speed
Accuracy Bonus: 5-15 exp based on accuracy
Level Up: Every 100 experience points

😊 Happiness System

High Performance: +5 happiness
Low Performance: -3 happiness
Feeding: +5 happiness
Playing: +10 happiness

🎨 Animation System

Each pet evolution stage has unique sprite sheets with multiple animation states:

// Animation states for each pet: const animations = { idle: [0, 1, 2, 3], // Gentle breathing/movement walkLeft: [4, 5, 6, 7], // Walking left animation walkRight: [8, 9, 10, 11], // Walking right animation happy: [12, 13, 14, 15], // Happy/excited animation sad: [16, 17, 18, 19] // Sad/disappointed animation };
⚠️ Note: You'll need to create or obtain sprite sheets for each evolution stage. Check the /public/sprites/README.md file for detailed requirements and examples.

🚀 Getting Started

To use the pet system in your typing trainer:

  1. Import the Pet component: import { Pet } from './components/Pet';
  2. Add it to your page: <Pet onPetInteraction={handleInteraction} />
  3. Update pet after tests: petManager.updatePetFromTestResult(result);
  4. Create sprite sheets: Follow the README guidelines in the sprites folder
✅ Success! The pet system is now fully integrated into your typing trainer! Your users will have a fun, engaging way to track their progress and stay motivated.

🔧 Technical Details

The pet system is built with:

  • TypeScript: Fully typed interfaces and classes
  • Canvas API: Hardware-accelerated sprite rendering
  • Local Storage: Persistent pet state across sessions
  • Singleton Pattern: Centralized pet management
  • Responsive Design: Works on all screen sizes
  • Animation Loop: Smooth 60fps animations
